# CShark
This is project of assignment 6 of POSTECH CSED353 Network class.

## Build Essentials
- libpcap
- CMake 2.5 or higher
- Make
- g++ with C(++) Standard 11

## How to build
```shell
$ rm -rf CMakeFiles
$ rm -rf CMakeCache.txt
$ cmake .
$ make
```

## How to execute
```shell
$ ./CShark
```
